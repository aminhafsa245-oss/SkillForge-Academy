# 🎓 SkillForge Academy

A modern and responsive Training & Internship Portal built using **HTML, CSS, and JavaScript**.

## 📖 Overview

SkillForge Academy is a professional learning platform designed to help students develop industry-ready skills through training programs, internships, and career development opportunities.

The website provides an attractive and user-friendly interface where users can explore programs, internship opportunities, industry partnerships, and apply online.

## ✨ Features

* Responsive Modern Design
* Professional Landing Page UI
* Training Programs Section
* Internship Opportunities Section
* Industry Partners Showcase
* Student Success Stories
* Application Form
* Mobile-Friendly Navigation
* Smooth Scrolling Effects
* Interactive Animations
* Premium Dark Theme

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
